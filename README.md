Prerequisites:  

- [mediainfo](https://mediaarea.net/en/MediaInfo)

Dependencies for bot core:

- Python IRC module (irc)
- termcolor

Dependencies for link handler (Main Feature - Recommand):

- requests
- Pillow
- BeautifulSoup4
- html5lib


Manual Usage:
```
git clone https://github.com/BruceZhang1993/PyIrcBot
cd PyIrcBot
python3 -m pyircbot
```
